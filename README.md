# Simulador de pontos

Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos de operadores aritmeticos no javascript.


## Aprendizados

Compreendi os conceitos de operadores aritmeticos no javascript.
## Funcionalidades

- Ao clicar nos botoes voce podera ganhar e perder pontos.

## Stack utilizada

**Front-end:** Html, css & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/gerador-de-nomes
```
